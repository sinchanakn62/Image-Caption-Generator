Image Caption Generator

Description
This project generates meaningful captions for images using Deep Learning using CNN and LSTM models.

Technologies Used
- Python
- TensorFlow / Keras
- Deep Learning
- Flask
- CNN + LSTM

Features
- Upload image
- Automatically generate caption
- Simple web interface

Applications
- Assist visually impaired users
- Image understanding systems
- AI-based automation

Future Scope
- Improve accuracy using transformers
- Add voice output
- Deploy using cloud
